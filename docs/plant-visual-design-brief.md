# 🌿 습관 정원 — 식물 비주얼 디자인 의뢰서

> 외부 디자이너용 커미션 문서. 이 문서 하나로 시안을 제작할 수 있도록 필요한 정보를 담았습니다.
> 한 번에 전체를 그리지 않고, **대표 6종 스타일 시안 → 방향 확정 → 등급별 파일럿 → 전체 제작** 순으로 진행합니다.

---

## 1. 프로젝트 개요

**"습관 정원"** 은 습관·기도를 기록하면 정원의 식물이 자라고 만개해 보상을 주는 모바일/웹 앱입니다. 기독교 신앙을 테마로 한 **따뜻하고 희망적인** 무드예요. 식물은 **씨앗 → 성장 단계 → 만개**까지 자라며, 사용자가 꾸준할수록 정원이 풍성해집니다.

본 의뢰는 **식물 종의 비주얼**(성장 단계별 프레임) 제작입니다.

---

## 2. 작업 범위

- 식물 종 × 각 종의 **성장 단계별 프레임** (종마다 2~8프레임 — §6 표의 '단계' 열).
- 각 종의 핵심은 **만개(최종 프레임)** — 가장 아름답고 보상감 있게.
- **6등급 시각 위계**: 기본(소박) → 신성(영광).
- (옵션) 시듦 상태 처리 — §5.

> **물량 안내:** 최종 채택은 **약 40종**이지만, **후보 풀(약 58종, §6-B)** 을 넉넉히 그려 주시면 그 중 잘 나온 것을 추려 최종 로스터를 확정합니다. 후보 중 일부만 제작할지, 전부 제작할지는 §10 단계에 따라 협의합니다.

---

## 3. 아트 디렉션

무드: **코지(cozy)·생명력·희망**, 고티어는 **경건·영광**.

**스타일 방향 후보(시안 탐색용)** — 대표 6종으로 A/B/C를 제안해 주시면 한 방향을 선택합니다:

- **A. 플랫·미니멀 벡터** — 밝은 색면, 가는 선. 작은 그리드에서 또렷.
- **B. 세미플랫 동화풍** — 부드러운 그라데이션·약한 음영. 따뜻함.
- **C. 보태니컬 일러스트** — 섬세한 선·수채 느낌. 우아·고급.

원칙: **실루엣 우선**(작은 사이즈에서도 종 식별), 전 종이 **공통 규격·바닥선·시점(정면에서 약간 위)·광원(좌상단)** 을 공유.

---

## 4. 등급별 시각 언어

| 등급 | 무드 | 색·광 |
|---|---|---|
| 기본 basic | 풋풋·소박 | 채도 낮은 초록, 광효과 없음 |
| 일반 common | 친근·밝음 | 맑은 색, 아주 옅은 광 |
| 희귀 rare | 인상적 | 또렷한 색, 은은한 글로우 |
| 에픽 epic | 신비(성경) | 깊은 보석색, 뚜렷한 글로우 |
| 전설 legendary | 위엄·신비 | 금/심홍/향 연기, 강한 아우라 |
| 신성 sacred | 거룩·영광 | 백금/황금 후광, 신성한 광채 |

> 앱이 **등급별 글로우를 코드로 자동 부여**합니다 → 아트 자체는 깔끔하게(과한 외곽광 지양). 단 **의도된 특수효과(불꽃·후광·향 연기)는 아트에 포함**해 주세요. 글로우 색 가이드는 별첨 가능.

---

## 5. 성장 단계 & 상태

- **0단계**: 흙 속 씨앗/새싹(작게) → **중간**: 키·잎·줄기 점증(같은 식물이 자라는 연속성) → **최종**: 만개(절정·보상감).
- 단계는 **실루엣·키로 또렷이 구분**. 바닥선·화분 위치는 고정.
- **상태**: 앱은 정원 '생기'가 낮으면 식물을 **시든 모습**으로 표시합니다. 최소한 만개 프레임의 *시든 변형*, 또는 시듦 처리 가이드(채도↓·잎 처짐)가 필요합니다. (전 단계 시든 변형까지 할지는 협의)

## 7. 기술 사양

- **포맷**: 벡터 **SVG 우선**(AI/SVG 레이어 분리). 개발이 SVG로 통합. 미리보기 **PNG 동봉**.
- **캔버스**: 정사각 viewBox(예 256×256), **투명 배경**. 식물 밑동을 **하단-중앙 기준선(y≈85%)** 에 정렬.
- 종별·단계별 **개별 아트보드**. 같은 종은 동일 캔버스·정렬·스케일 유지.
- 팔레트 제한(종당 ~4~6색), 등급 색 가이드 준수. **라스터 임베드 금지**, 텍스트 아웃라인화.
- **표시 크기**: 도감 그리드 ~48~64px(작게) + 정원 확대(크게) 양쪽에서 읽히게.
- **애니메이션**: 기본 정적 (추후 만개 시 미세 idle 흔들림은 옵션으로 협의).

## 8. 파일·네이밍 규칙

- `{id}_stage{n}.svg` — n = 0 … 최종(최종 = 만개). 예: `fig_stage0.svg` … `fig_stage4.svg`.
- 종별 폴더 `/{id}/`. **영문 id 고정**(§6 표) — 개발 연동 키이므로 변경 금지.

## 10. 진행 · 검수 프로세스

1. **스타일 탐색**: 대표 6종(`sprout` · `sunflower` · `maple` · `fig` · `cedar` · `tree_of_life`) × A/B/C 방향 시안
2. 방향 확정 →
3. **등급별 파일럿**(티어당 1종 풀 단계) 검수 →
4. **후보 풀 제작** 후 채택분 추림 → 최종 ~40종 풀 단계 →
5. 시듦·특수효과 마감
- 산출물: 소스(SVG/AI) + PNG 프리뷰 + 네이밍 준수.
